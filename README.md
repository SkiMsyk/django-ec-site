# Create django project

```
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install django==3.1.12
$ mkdir vegeket
$ cd vegeket
$ django-admin startproject config .
```

# Build docker environment

```
$ docker-compose build
$ docker-compose up
$ docker-compose exec web bash
```

then you will be in bash.
